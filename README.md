# Premium Tasbeeh UI

A mobile-friendly, premium-styled Tasbeeh web app built with static HTML, CSS, and JavaScript.

## Features

- Multi-screen welcome flow
- Counter progress and step guidance
- Responsive layout for mobile devices
- Glassmorphism and premium visual styling

## Run locally

1. Open `open.html` in your browser.
2. Or use a local server:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000/open.html`.

## Files

- `open.html` — main app page
- `static/styles.css` — app styling
- `static/app.js` — page flow logic
- `index.html` — marketing homepage
- `app.py` — Flask fallback server
